#### 练习

##### 查询"01"课程比"02"课程成绩高的学生的信息及课程分数  

```sql
select stu.*, sc1.c_id as '语文', sc1.s_score, sc2.c_id as '数学', sc2.s_score
from Student stu
left join Score sc1 on sc1.s_id=stu.s_id and sc1.c_id='01'
left join Score sc2 on sc2.s_id=stu.s_id and sc2.c_id='02'
where sc1.s_score > sc2.s_score
```

##### 查询"01"课程比"02"课程成绩低的学生的信息及课程分数 

```sql
select stu.*, sc1.c_id as '语文', sc1.s_score, sc2.c_id as '数学', sc2.s_score
from Student stu
left join Score sc1 on sc1.s_id=stu.s_id and sc1.c_id='01'
left join Score sc2 on sc2.s_id=stu.s_id and sc2.c_id='02'
where sc1.s_score < sc2.s_score
```

##### 查询平均成绩大于等于60分的同学的学生编号和学生姓名和平均成绩 

```sql
select sc.s_id, AVG(s_score) as avg_score , st.*
from Score sc
left join Student st on sc.s_id=st.s_id
group by sc.s_id having avg_score>60
```

##### 查询平均成绩小于60分的同学的学生编号和学生姓名和平均成绩(包括有成绩的和无成绩的) 

```sql
select sc.s_id, AVG(s_score) as avg_score , st.*
from Score sc
left join Student st on sc.s_id=st.s_id
group by sc.s_id having avg_score<60 or  avg_score is null
```

##### 查询所有同学的学生编号、学生姓名、选课总数、所有课程的总成绩 

```sql
select st.s_id, st.s_name, COUNT(DISTINCT sc.c_id) as course_cnt, SUM(case when sc.s_score is NULL then 0 else sc.s_score end) as course_sum 
from Student st
left join Score sc on sc.s_id=st.s_id
GROUP BY st.s_id
```

##### 查询"李"姓老师的数量 

```sql
select t_name, count(t_id) 
from Teacher 
group by t_name having t_name like '李%'
```

##### 查询学过"张三"老师授课的同学的信息

```sql
select st.* 
from Score sc
left join Student st on sc.s_id=st.s_id
left join Course co on sc.c_id=co.c_id
left join Teacher te on co.t_id=te.t_id
where te.t_name='张三'
```

##### 查询没学过"张三"老师授课的同学的信息

```sql
select st.s_id, st.`s_name`, st.`s_birth`, st.s_sex, group_concat(te.t_name) as all_tes
from Student st 
left join Score sc on sc.s_id=st.s_id
left join Course co on sc.c_id=co.c_id
left join Teacher te on co.t_id=te.t_id
group by st.s_id, st.`s_name`, st.`s_birth`,st.s_sex HAVING all_tes not like '%张三%' or all_tes is NULL
```
